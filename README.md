HoloLens 2 XR Fluid & Rope Interaction System

프로젝트 소개

Microsoft HoloLens 2 기반 혼합현실(MR) 환경에서 사용자 인터페이스를 설계 및 구현하고, 무선 환경에서 물리 기반 유체 및 로프 시뮬레이션을 실시간으로 제어할 수 있는 인터랙션 시스템을 개발하였다.
본 프로젝트는 XR 환경에서의 자연스러운 물리 표현과 직관적인 사용자 조작 경험 제공을 목표로 한다.

주요 구현 기능

HoloLens 2 기반 MR 인터랙션 시스템
HoloLens 2 환경에서 동작하는 사용자 인터페이스 설계 및 구현
공간 인식 기반 입력을 활용한 MR 인터랙션 구조 구성
헤드 트래킹 기반 시점 중심 인터랙션 처리

외력 기반 유체 및 로프 물리 시뮬레이션
외부 입력(Force)에 반응하는 유체 및 로프 물리 시스템 구현
물리 기반 연산을 활용한 자연스러운 변형 및 움직임 구현
실시간 파라미터 제어를 통한 동적 시뮬레이션 변화 지원

무선 제어 기반 실시간 인터랙션 구조
HoloLens 디바이스를 통한 무선 입력 기반 제어 시스템 설계
네트워크/입력 지연을 고려한 실시간 반응 구조 적용
물리 시뮬레이션과 사용자 입력 간 동기화 처리

XR 환경 최적화 및 실시간 렌더링 구조
혼합현실 환경에서의 성능 제약을 고려한 경량화 설계
실시간 로프 시뮬레이션을 위한 업데이트 구조 최적화
지속적인 인터랙션에도 안정적인 프레임 유지 구조 적용
